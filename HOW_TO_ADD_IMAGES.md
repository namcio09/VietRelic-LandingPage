# Hướng dẫn đưa ảnh vào Hero Carousel

## Bước 1: Chuẩn bị ảnh

Bạn cần **6 ảnh** (hoặc ít nhất 3 ảnh banner):
- **3 ảnh banner lớn** (cho slide chính)
- **3 ảnh phụ** (cho phần bên phải)

## Bước 2: Copy ảnh vào project

### Cách A: Copy trực tiếp vào thư mục

1. Mở File Explorer (Windows) hoặc Finder (Mac)
2. Điều hướng đến thư mục:
   ```
   vietrelic-landingPage/src/assets/images/
   ```
3. Copy ảnh vào thư mục này

**Đặt tên ảnh như sau:**
- `hero-banner-1.jpg` (hoặc .png) - Banner slide 1
- `hero-banner-2.jpg` - Banner slide 2
- `hero-banner-3.jpg` - Banner slide 3
- `hero-side-1.jpg` - Ảnh phụ slide 1
- `hero-side-2.jpg` - Ảnh phụ slide 2
- `hero-side-3.jpg` - Ảnh phụ slide 3

### Cách B: Drag & Drop vào Cursor

1. Mở Cursor IDE
2. Mở sidebar **Explorer** (Ctrl+Shift+E)
3. Tìm thư mục `src/assets/images/`
4. **Kéo thả** ảnh từ máy tính vào thư mục này
5. Đổi tên ảnh theo format trên

## Bước 3: Thông báo cho tôi

Sau khi bạn đã copy ảnh vào thư mục, **nói cho tôi biết**:
- Tên các file ảnh bạn đã đặt
- Hoặc chỉ cần nói "đã copy xong" và tôi sẽ tự động tìm và cập nhật

## Bước 4: Tôi sẽ cập nhật code

Tôi sẽ:
1. Tìm ảnh trong thư mục `src/assets/images/`
2. Cập nhật `Hero.tsx` để import và sử dụng ảnh thật
3. Test để đảm bảo ảnh hiển thị đúng

## Lưu ý

- **Định dạng ảnh**: JPG, PNG, WebP đều được
- **Kích thước khuyến nghị**:
  - Banner: 1200x800px hoặc lớn hơn
  - Ảnh phụ: 600x400px hoặc lớn hơn
- **Tên file**: Có thể đặt tên khác, chỉ cần cho tôi biết tên là gì

## Nếu chỉ có 3 ảnh banner

Nếu bạn chỉ có 3 ảnh banner (không có ảnh phụ), tôi sẽ:
- Dùng cùng ảnh banner cho cả banner và ảnh phụ
- Hoặc tạo ảnh phụ từ banner (crop/resize)

---

**Bắt đầu:** Copy ảnh vào `src/assets/images/` và cho tôi biết khi xong!
